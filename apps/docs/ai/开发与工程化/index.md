# 开发与工程化

AI开发与工程化涵盖了AI应用开发过程中的工程实践、开发框架和方法论，帮助开发者高效构建、测试和维护AI系统。

## 主要内容

- **Prompt工程** - 设计高效prompt的方法、技巧和最佳实践
- **开发框架** - LangChain、LlamaIndex等AI应用开发框架的使用
- **工程最佳实践** - AI应用的测试、版本管理、持续集成/部署
- **开发工具链** - IDE插件、调试工具、性能分析工具
